## 프로젝트 소개
Reactjs에서 상태관리를 위해 사용하는 Redux를 공부하고 적용해보기 위해 혼자서 진행한 프로젝트입니다. 평소에 즐겨보던 Netflix의 홈화면을 구현해보았습니다.영화의 상세정보를 확인할수 있는 기능을 구현하기 위하여 The Movie DataBase API를 사용했습니다.
